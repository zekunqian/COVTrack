# COVTrack: Continuous Open-Vocabulary Tracking via Adaptive Multi-Cue Fusion

## Environment Setup
Please refer to the OVTrack installation guide: [OVTrack Installation](https://github.com/SysCV/ovtrack/blob/main/docs/INSTALL.md)

## Dataset
The dataset used in the `data` directory can be referenced from OVTrack: [OVTrack Get Started](https://github.com/SysCV/ovtrack/blob/main/docs/GET_STARTED.md)

## Provided Models
The following models are included in the `saved_models` directory, which is downloaded from [COVTrack HuggingFace](https://huggingface.co/clarkqian/COVTrack):

- **ctao_dataset/ctao_base.json**: Annotations for the CTAO dataset (only base categories, used in COVTrack).
- **ctao_dataset/ctao_base_and_novel.json**: Annotations for the CTAO dataset (both base and novel categories).
- **ctao_public_res/ctao_public.pth**: Weights for the results of "Ours" from the CTAO paper.
- **pretrained_models/detpro_prompt.pt**: CLIP text embeddings for 1203 classes obtained using DetPro.
- **pretrained_models/ovtrack_pair.pth**: Model trained using image pairs generated by the diffusion model provided by OVTrack.
